📦 Daftar - Backend API

Daftar is a powerful backend API built using Node.js, Express, and MongoDB for managing invoices, products, customers, stock, and detailed reports.

🔧 Features

  • Authentication (JWT based)
  • Users & Roles Management
  • Customer Management
  • Product & Inventory Management
  • Invoice Management with payment tracking
  • Reports: Daily Revenue, Monthly Sales, Unpaid Invoices, etc.

🚀 Technologies Used

  • Node.js & Express
  • MongoDB & Mongoose
  • JWT for authentication
  • MVC architecture
  • Postman for API testing

📂 Folder Structure

  • /controllers – Logic for handling each route
  • /routes – API endpoints
  • /models – Mongoose schemas
  • /middleware – Auth and error handling
  • /utils – Helper functions

📮 Main API Endpoints

  • POST /api/auth/register – User registration
  • POST /api/auth/login – User login
  • GET /api/users – Get all users
  • GET /api/customers – Get customers
  • POST /api/invoices – Create invoice
  • GET /api/reports/daily-revenue – Daily revenue report
  • ...and more

📁 Database

MongoDB collections include:

  • Users
  • Customers
  • Products
  • Invoices
  • InventoryTransactions

✅ Usage

  1. Clone the repository
  2. Run npm install
  3. Configure .env with Mongo URI and JWT secret
  4. Run npm run dev to start the server
